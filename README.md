# Mobile Data Explorer

## Project Overview
Mobile Data Explorer is a data analysis project that deals with the exploration of mobile phone sales data to determine trends in brand popularity, price, and customer preferences. The project involves analyzing the sales data and interpreting the results in an interactive manner.

## Tools & Technologies
- Python
- SQL
- Power BI
- Pandas
- NumPy
- Excel

## Project Objectives
The project aims to:
- Analyze mobile sales data to determine the top-selling brands.
- Determine the price trends in various mobile phone models.
- Explore customer preferences and purchase habits.
- Create interactive dashboards to effectively visualize the data.

## Data Analysis Process
The data analysis process involves:
1. Data Collection and Loading.
2. Data Cleaning and Preprocessing Using Python.
3. Exploratory Data Analysis.
4. Data Visualization Using Power BI Dashboards.

## Key Insights
The key insights obtained from the project are:
- Identified popular mobile brands.
- Analyzed price trends in various mobile phone models.
- Visualized customer preferences and purchase habits.
